# Predictive Sales Forecast

## Problem Statement
The objective of this project is to predict future sales using historical sales data.

## Dataset
The dataset contains monthly sales records.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Jupyter Notebook
- GitHub

## Methodology

### 1. Data Collection
Collected monthly sales data.

### 2. Data Analysis
Analyzed sales trends using Pandas.

### 3. Data Visualization
Created graphs to visualize sales growth.

### 4. Model Building
Used Linear Regression to train the forecasting model.

### 5. Prediction
Predicted future sales for upcoming months.

## Results
- Generated sales trend graphs.
- Forecasted future sales successfully.
- Evaluated model performance using R² Score.

## Conclusion
The project demonstrates how machine learning can be used to forecast future sales and support business decision-making.
